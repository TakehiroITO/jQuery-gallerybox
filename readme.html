XML

<?xml version="1.0" encoding="utf-8"?>
<gallryBox>
	<component start="2012/12/25" end="2012/12/27">
		<item num="1">
			<img>images/image1.jpg</img>
			<url>#test1</url>
			<alt>ALT TEXT</alt>
			<thumb>images/thumb1.jpg</thumb>
		</item>
		<item num="2">
			<img>images/image2.jpg</img>
			<url>#test1</url>
			<alt>ALT TEXT</alt>
			<thumb>images/thumb2.jpg</thumb>
		</item>
	</component>
</gallryBox>

itemの属性numは管理用に記載しているので、省略しても問題ありません。

urlは省略できます。画像にリンクが不要な場合は削除してください。

thumbは省略できます。サムネイルをXMLで指定したい場合、javascript側と併せて記述してください（上記参照）。

XMLのcomponentの属性は startと endです。
属性start endは省略可能です。

startは表示開始、endは表示終了の日付です。
書式は yyyy/MM/dd です。

<component start="0000/00/00">

それぞれ時刻も指定できます。
書式は yyyy/MM/dd HH:mm:ss です。

<component start="0000/00/00 00:00:00">

componentは複数置くことができます。

<gallryBox>
	<component>
		<item>
			<img>images/image1.jpg</img>
			<url>#test1</url>
			<alt>ALT TEXT</alt>
			<thumb>images/thumb1.jpg</thumb>
		</item>
	</component>
	<component>
		<item>
			<img>images/image1.jpg</img>
			<url>#test1</url>
			<alt>ALT TEXT</alt>
			<thumb>images/thumb1.jpg</thumb>
		</item>
	</component>
</gallryBox>

この場合、表示可能になっている画像はcomponentをまたいで表示されます。

動作

画像が1枚のとき、画像だけ表示し、サムネイル無し
画像が2〜3枚のとき、サムネイルを表示して、表示されている画像に対応したサムネイルへマーカーが移動します。
画像が4枚のとき、サムネイルを3枚表示してスクロールアニメーションします。
画像が5枚以上のとき、サムネイルを5枚表示してスクロールアニメーションします。
